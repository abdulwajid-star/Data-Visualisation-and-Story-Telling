# 📊 Superstore Sales Performance Dashboard (Power BI)

## Project Overview

This project presents an interactive **Sales Performance Dashboard** developed using **Microsoft Power BI** with the Sample Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, product categories, and regional sales through interactive visualizations.

The objective is to transform raw sales data into meaningful business insights using data visualization and storytelling techniques.

---

## Objectives

- Analyze sales and profit performance
- Identify top-performing product categories and sub-categories
- Visualize sales trends over time
- Analyze regional sales distribution
- Evaluate customer segments and shipping modes
- Demonstrate data storytelling using interactive dashboards

---

## Tools & Technologies

- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- Sample Superstore Dataset

---

## Dataset

**Dataset:** Sample Superstore

The dataset contains information about:

- Orders
- Sales
- Profit
- Discount
- Quantity
- Customer Segment
- Product Category
- Region
- State
- Shipping Mode

---

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Average Order Value

### Interactive Filters
- Region
- Category
- Segment
- Order Date

### Visualizations
- Monthly Sales Trend (Line Chart)
- Sales by Category (Clustered Column Chart)
- Profit by Sub-Category (Clustered Bar Chart)
- Sales by State (Filled Map)
- Sales by Customer Segment (Donut Chart)
- Sales by Ship Mode (Column Chart)
- Discount vs Profit (Scatter Chart)

---

## Key Business Insights

- Technology category generates the highest sales.
- Consumer segment contributes the largest share of revenue.
- Sales show an overall upward trend over time.
- Some product sub-categories generate lower or negative profits.
- Regional analysis highlights the states with the highest sales.
- Higher discounts are often associated with lower profitability.
